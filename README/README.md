# RefugisLliures_Frontend

A React Native application built with Expo.

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Expo Go app on your mobile device (for testing)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/jordirev/RefugisLliures_Frontend.git
cd RefugisLliures_Frontend
```

2. Install dependencies:
```bash
npm install
```

## Running the App

Start the development server:
```bash
npm start
```

This will open Expo Developer Tools in your browser. From here you can:

- Press `a` to open on Android emulator
- Press `i` to open on iOS simulator (macOS only)
- Press `w` to open in web browser
- Scan the QR code with Expo Go app on your mobile device

### Platform-specific commands

Run on Android:
```bash
npm run android
```

Run on iOS (macOS only):
```bash
npm run ios
```

Run on web:
```bash
npm run web
```
